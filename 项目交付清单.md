## 📝 使用说明

### 快速启动

```bash
# 1. 安装依赖
npm install

# 2. 启动开发服务器
npm run dev

# 3. 访问 http://localhost:3000
```

### 自定义修改

1. 修改 `config/site.ts` - 公司基本信息
2. 修改各页面内容 - 业务相关内容
3. 修改 `app/globals.css` - 主题颜色
4. 添加图片到 `public/` - 品牌资源
