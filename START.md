# 🚀 立即开始使用

## 只需3步，启动您的网站！

### 1️⃣ 安装

```bash
npm install
```

### 2️⃣ 启动

```bash
npm run dev
```

### 3️⃣ 访问

打开浏览器: **http://localhost:3000**
